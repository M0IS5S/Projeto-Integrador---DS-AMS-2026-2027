# Gerenciador de Cronograma Semestral

Uma aplicação Web e Mobile voltada para a construção de um cronograma de aulas na Fatec Itu. O objetivo principal do projeto é formalizar e facilitar a criação das aulas dentro do período de um semestre, contando com a parte Web, Mobile e também uma parte dedicada a disponibilidade do professor, garantindo um horário bem definido pelo coordenador e seguindo a disponibilidade cadastrada pelo professor para exibir o resultado aos alunos em prol da melhoria do ambiente escolar.
